def
Функцию можно определить при помощи ключевого слова def,за которым должно следовать название функции func и список её формальных параметров в круглых скобках (x,y). 
На следующих строках, выделенное отступом слева, должно располагаться тело функции. Для возврата значения из функции используется инструкция return. Допускается использование нескольких return, в том числе для раннего выхода из функции.
Пример: 
def func(x, y):
	Тело функции
	return x + y

range() - является универсальной функцией Python для создания списков (list) содержащих арифметическую прогрессию. Функция range() может принимать от одного до трех агрументов, при этом аргументами должны быть целые числа (int). range(start, stop, step) - так выглядит стандартный вызов функции range() в Python. По умолчанию start равняется нулю, step единице.
Чаще всего она используется в циклах for. Для повторения цикла некоторое заданное число раз n можно использовать цикл for вместе с функцией range:
Пример:
for i in range(n):
	Тело цикла

Функция destroy() уничтожает виджет w и все его дочерние элементы.
Пример:
w.destroy()

Функция append(). Добавляет элемент в конец списка list, x элемент.
Пример:
list.append(x)
